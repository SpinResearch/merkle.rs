An inclusion proof represent the fact that a value is a member of a MerkleTree with root hash root_hash, and hash function digest.

Fields

digest: D

The hash function used in the original MerkleTree

root_hash: Vec<u8>

The hash of the root of the original MerkleTree

lemma: Lemma

The first Lemma of the Proof

value: T

The value concerned by this Proof
